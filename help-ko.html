이건 뭔가요?

워프레임 클랜 도장 레이아웃을 계획하기 위한 도구입니다. 저는 게임 내 방 경계와 제약 조건을 최대한 정확하게 모델링하려고 노력했습니다.

주의사항

모든 방의 경계는 게임 내 실험, 웨이포인트 측정 또는 추측을 통해 세팅되어 있습니다. 문제가 있거나 오래되고 중단된 방의 경계와 통계에 대해서 도움을 주고 싶으시다면 저에게 연락해 주세요!

시작하기

기본적인 클랜 홀은 이미 추가되어 있습니다.

방 추가 버튼을 클릭하면 추가할 수 있는 방에 대한 카테고리가 있습니다. 방을 선택 후 배치하고 싶은 위치에 배치하세요.

키보드 단축키 치트시트

  • Shift+클릭: 클릭한 방을 선택
  • Shift+드래그: 드래그 영역내의 모든 방을 선택
  • Ctrl+Shift+드래그: 모든 층에 걸쳐 드래그 영역내의 모든 방을 선택
  • Ctrl+A: 현재 층의 모든 방 선택
  • Ctrl+Shift+A: 모든 레이아웃과 모든 층의 방을 선택
  • 배경 드래그: 화면을 스크롤
  • Ctrl +배경 드래그: 커서가 창 테두리 근처에 있을때 화면을 자동 스크롤
  • 방 드래그 + Shift: 방을 드래그 할 때 스냅(자석 모드) 비활성화
  • 방 드래그+Ctrl/Command: 8m이내의 가장 가까운 방으로 스냅
  • R: 선택한 방을 회전 (드래그 중에도 작동)
  • D: 방을 복사합니다
  • Alt+클릭: 방이나 선택된 방들을 삭제
  • Ctrl+C: 선택한 방 복제
  • Ctrl+X: 선택한 방 잘라내기
  • Ctrl+V: 복사하거나 잘라낸 방 붙여넣기
  • 위쪽 화살표: 시점을 한층 위로 이동
  • 아래쪽 화살표: 시점을 한층 아래로 이동
  • ESC: 방 추가/옮김을 취소하거나 열려있는 모든 메뉴 취소

네비게이션

이동하려면 격차 영역의 아무곳이나 클릭 후 드래그하세요.

방이 선택되어 있지 않으면 방을 클릭해서 드래그해도 화면이동이 가능합니다. (방 선택 참조).

방을 선택하고 다른 위치로 드래그하는 경우, 창 테두리 근처로 드래그하면 자동으로 해당 방향으로 스크롤됩니다. 커서가 창 테두리에 가까울수록 스크롤 속도가 빨라집니다.

확대/축소

확대하거나 축소하려면 마우스 휠을 사용하세요. 커서 위치를 중심으로 확대/축소 됩니다.

터치 스크린에서는 두 손가락을 모으거나 벌려서 확대/축소 합니다.

층 변경

화면 왼쪽에 강조 표시된 층은 현재 당신이 어떤 층을 보고 있는지 확인해줍니다. 층수는 1층부터 시작해서 위로 올라갑니다. 1층 아래는 B1이고 거기서부터 지하층이 시작됩니다.

다음 층을 보려면 층 목록 위에 있는 화살표 버튼을 클릭하세요. 아래층을 보려면 아래쪽 화살표 버튼을 클릭하세요.

키보드의 위쪽, 아래쪽 화살표 키를 사용하여 층 사이를 이동할 수도 있습니다.

해당 층의 버튼을 클릭해 해당 층으로 바로 이동됩니다.

층 목록에는 한 번에 표시할 수 있는 최대 층 수가 있습니다. 상단 또는 하단에 남아있는 층이 있는 경우, 화살표 버튼을 계속 눌러 표시되는 층을 바꿀수 있습니다.

오류가 있는 방이 포함된 층은 빨간색으로 강조 표시됩니다.

편집

방 선택

드래그하지 않고 클릭하여 방을 선택합니다. 현재 선택된 방 주위에는 파란색 윤곽선이 표시됩니다.

현재 층에 없지만 배경쪽에 회색으로 보이는 다른 층의 방을 클릭하면 해당 층으로 바로 이동하면서 방이 선택됩니다. 클릭하려는 방 바로 위층에 다른 방이 있는 경우, 가장 높은 층에 있는 방이 선택됩니다.

다중 선택

다중 선택 모드를 활성화하려면 Shift키를 누르거나 다중 선택 버튼을 클릭하세요.

다중 선택이 활성화된 동안 방을 클릭하면 해당 방이 현재 선택에 추가됩니다. 이미 선택된 방을 클릭하면 선택이 취소됩니다.

다중 선택이 활성화된 동안 클릭하고 드래그하면 선택 상자가 생성됩니다. 이후 손을 떼면 선택 상자의 모든 방이 현재 선택 항목에 추가됩니다.

기본적으로 다중 선택 모드는 현재 층의 방만 선택합니다. Shift키와 Ctrl/Command를 누른 상태에서 방을 드래그하거나 클릭하면 모든 층이 선택됩니다. 다중 층 선택이 활성화되면 다른 다중 선택 아이콘인 다중 선택이 표시됩니다.

모두 산택

현재 층의 모든 방을 선택하려면 Ctrl+A를 사용하세요.

모든 층에 걸쳐 전체 레이아웃의 모든 방을 선택하려면 Shift+Ctrl+A 또는 Shift+Command+A를 사용하세요.

방 메뉴에서 모두 선택을 사용하여 도장 전체에서 동일한 유형의 모든 방을 선택합니다.

방 추가

방 추가 버튼을 클릭하면 각 카테고리 메뉴가 표시됩니다. 이 카테고리 중 하나를 클릭하면 방이 표시됩니다. 이 중 하나를 클릭하면 방이 커서 아래에 붙습니다. 이제 방을 배치할 곳에 옮긴 후 클릭하세요.

이미 한 번 방을 선택하거나 배치한 경우에는 방 추가 메뉴 하단에 마지막으로 선택하거나 배치한 방이 표시됩니다.

배치된 방을 클릭하면 방 메뉴가 나타납니다. 이때 복제를 클릭하면 방의 복사본을 만들어 다른 곳에 배치할 수 있습니다.(복사는 복사 및 붙여넣기 참조)

방을 선택하고 D 키를 누르면 메뉴를 불러올 필요없이 방이 복제됩니다.

새로 생성된 방을 옮기는 동안 Esc 키를 누르면 추가 작업이 취소되고 방이 제거됩니다.

방 옮기기

하나 이상의 방을 선택한 후, 선택한 방 중 하나를 클릭하고 드래그하여 이동하세요.

Ctrl/Command를 누른 상태에서 방을 드래그하면 8m이내의 다른 방의 문에 위치를 맞춥니다. 누르지 않으면 가장 가까운 곳에 위치를 맞춥니다.

여러개의 방이 선택되어있는 경우, 한 방을 드래그해서 옮길 때 문 중 하나가 다른 방에 가까워지면 선택되어있는 모든 방들이 같이 이동되며 스냅됩니다.

  • 스냅의 민감도 영역은 확대/축소에 따라 변경되지 않습니다. 방을 올바른 문에 맞추는데 어려움이 있다면 확대해 보세요.
  • 만약 다른 문에 붙여야 할 문을 좀 더 세밀하게 제어하고 싶다면, 방을 드래그할때 붙이려고 하는 문에 가까운 영역을 클릭해서 드래그해보세요.

스냅을 비활성화하려면 Shift 키를 누른 채 방을 드래그하세요.

복사 붙여넣기나 방을 클릭해서 추가할떄도 Shift+드래그, Alt+드래그 기능을 사용할 수 있습니다.

하나 이상의 방을 드래그하는 동안 Esc 키를 누르면 작업이 취소되고 방이 시작된 곳으로 돌아갑니다.

방 회전시키기

현재 선택된 방을 다시 클릭하면 메뉴가 나타납니다. 회전을 클릭하면 방이 90도 회전합니다.

R 키를 눌러 선택한 방을 회전할 수도 있습니다. 이는 방을 드래그하는 동안에도 작동합니다.

두개 이상의 방을 선택한 상태에서는 모든 방이 회전됩니다. 회전의 중심은 회전을 사용한 방식에 따라 달라집니다.

  • 방 메뉴의 옵션에서 회전 버튼을 눌러 기능을 사용할 경우, 메뉴를 불러오기 위해 클릭한 방이 회전의 중심이 됩니다.
  • 방 여러개를 드래그 선택해서 회전 기능을 사용하면, 드래그 작업을 시작하기 위해 클릭한 방이 회전의 중심이 됩니다.
  • R 키를 사용하는 경우 회전 중심점은 선택된 방의 경계 상자입니다.

엘리베이터 층 변경

엘리베이터는 2개 층에 걸쳐 있으며, 현재 층에서 올라가거나 내려갑니다. 엘리베이터가 가는 방향을 전환하려면 해당 엘리베이터만 선택한 다음 다시 클릭하여 객실 메뉴를 불러옵니다. 그리고 층 변경을 선택합니다.

방 삭제

현재 선택된 방을 다시 클릭하면 방 메뉴가 나타납니다. 선택한 방을 삭제하려면 삭제를 클릭하세요.

메뉴를 불러올 필요 없이 선택한 방을 삭제하려면 Delete키를 누르세요.

선택하지 않은 방을 Alt 키를 누른 채 클릭하면 한 번의 클릭으로 삭제할 수 있습니다.

Alt 키를 누른 채 다중 선택된 방 중 하나를 클릭하면 한 번의 클릭으로 선택한 모든 방이 삭제됩니다.

복사 및 붙여넣기

하나 이상의 방을 선택한 상태에서 선택한 방을 클릭하고 방 메뉴의 복사 복사를 클릭하여 선택한 방을 내부 클립보드에 복사합니다. Ctrl+C를 사용할 수도 있습니다.

선택한 공간을 내부 클립보드에 복사하고 원본을 삭제하려면 메뉴의 잘라내기를 클릭하거나 Ctrl+X를 사용하세요.

클립보드에 방을 복사할 경우, 방 추가 메뉴에 아래에 붙여넣기 옵션이 있습니다. Ctrl-P를 사용할 수도 있습니다.

모두 삭제하기

메뉴를 클릭한 다음 새로 만들기를 클릭하여 새로 시작하세요.

새로운 링크를 여는 기능이므로, 브라우저의 뒤로 버튼을 사용하여 이전 레이아웃으로 돌아갈 수 있습니다

실행 취소 및 다시 실행

실행 취소를 클릭하거나 Ctrl+Z를 사용하여 마지막 작업을 실행 취소하세요. 작업에는 이동, 회전, 층 변경, 선택 항목 추가, 삭제 및 변경이 포함됩니다. 계속해서 실행 취소를 클릭하거나 Ctrl+Z를 사용하여 작업들을 실행 취소할 수 있습니다.

방금 취소한 작업을 다시 실행하려면 다시 실행을 클릭하거나 Ctrl+Shift+Z 또는 Ctrl+Y를 사용하세요. 계속해서 실행 취소를 클릭하거나 Ctrl+Shift+Z/Control+Y를 사용하여 이전에 실행 취소한 작업을 다시 실행할 수 있습니다.

실행 취소 후 다른 작업을 수행하면 다시 실행 가능한 작업이 모두 지워집니다.

최대 250개의 작업을 실행 취소할 수 있습니다.

꾸미기

게임에 없는 옵션이 포함된 몇 가지 장식 옵션을 사용할 수 있습니다. 핵심 레이아웃 메커니즘이나 규칙에는 영향을 미치지 않습니다. 이는 단지 레이아웃을 더 쉽게 만들고 이해할 수 있도록 하기 위한 것입니다.

방을 복제하거나 복사하면 코스메틱 사용자 정의가 그대로 유지됩니다.

색상

기본적으로 방은 게임 내 지도에 나타나는 것처럼 회색입니다. 그러나 객실 메뉴에는 색상 옵션이 있습니다. 그러면 선택한 모든 방에 색상을 할당할 수 있는 색상 선택기가 나타납니다.

세 개의 슬라이더를 사용하여 색조, 채도 및 밝기 를 조정합니다.

선택한 룸을 다시 회색으로 재설정하려면 지우기 를 클릭하세요.

저장을 클릭하거나 색상 선택기를 닫아 색상 변경 사항을 저장합니다.

색상 선택기를 닫고 재설정 및 변경하려면 취소 를 클릭하세요.

모두 선택 옆에는 비슷한 색상의 방을 선택할 수 있는 네 가지 옵션이 있습니다.

  • 모든 항목 일치: 동일한 색조, 채도 및 휘도 설정을 가진 도장의 모든 방을 선택합니다.
  • : 다른 색상 설정에 관계없이 동일한 색조를 가진 도장의 모든 방을 선택합니다.
  • : 다른 색상 설정과 상관없이 동일한 채조를 가진 도장의 모든 방을 선택합니다.
  • : 다른 색상 설정과 상관없이 동일한 휘도를 가진 도장의 모든 방을 선택합니다.

레이아웃에 색상이 있는 일부 공간이 이미 있는 경우, 해당 색상은 색조, 채도, 휘도 슬라이더에 마커를 생성합니다. 슬라이더를 마크에 충분히 가깝게 이동하면 마크에 맞춰집니다. Shift 키를 누르고 있으면 스냅을 비활성화할 수 있습니다.

현재 색상 설정이 다른 여러 방을 선택하고 수정하려는 경우 수정된 색상 부분만 변경됩니다. 예를 들어 빨간색과 녹색 방을 선택하고 색조를 건드리지 않고 휘도 슬라이더만 조정하면 방은 여전히 빨간색과 녹색으로 유지되며 밝기만 변경됩니다.

라벨

모든 방에는 라벨을 붙일 수 있습니다. 방 메뉴 아래에는 라벨 편집기를 불러오는 라벨 라벨 옵션이 있습니다. 방 하나를 선택한 경우에만 이용 가능합니다.

텍스트 상자에 방의 라벨을 입력합니다. 라벨은 여러 줄을 사용할 수 있습니다. 라벨은 선택한 방의 중심에 배치됩니다.

라벨의 크기를 변경하려면 눈금 눈금 슬라이더를 사용하십시오.

색상 선택기와 마찬가지로 레이아웃에 있는 다른 라벨의 크기는 눈금 슬라이더가 스냅할 수 있는 마커로 표시됩니다. 스냅을 비활성화하려면 Shift 키를 누르세요.

선택한 방에서 라벨을 제거하려면 지우기를 클릭합니다.

저장을 클릭하거나 라벨 편집기를 닫아 라벨 변경 사항을 저장하세요.

취소를 클릭하면 라벨 편집기가 닫히고 초기화 및 변경됩니다.

방에 색상 사용자 정의가 있는 경우 해당 색상이 라벨에도 적용됩니다.

해당 라벨을 클릭하여 방을 선택하고 끌 수 있습니다.

라벨 객체

방 추가 메뉴의 유틸리티 폴더 안에는 라벨 객체가 별도로 있습니다.

라벨 객체는 리소스 총계 또는 도장 경계에 포함되지 않습니다. 때문에 도장방 중앙에 묶이지 않은 라벨을 배치할 수도 있습니다. 원하는 만큼 많이 배치할 수도 있습니다.

PNG를 생성할 때 이미지의 경계는 레이블이 아닌 실제 공간에 따라 결정됩니다. 레이아웃에서 벗어난 상태의 라벨은 표시되지 않습니다.

제한사항

방 배치에 대한 제한에는 리소스, 경계 및 규칙의 세 가지 유형이 있습니다.

방 자원

도장의 모든 방에 영향을 미치는 세 가지 제한 조건이 있습니다.

  • 방 수: 총 128개 객실로 제한됩니다.
  • 에너지:: 각 방에는 일정량의 에너지가 필요합니다. 사용 가능한 에너지를 늘리려면 원자로를 건설해야 합니다.
  • 수용량: 각 방에는 일정량의 수용량이 필요합니다. 더 많은 수용량을 얻으려면 클랜 홀이나 다른 홀을 건설해야 합니다.

도장의 현재 방 수, 에너지, 수용량이 왼쪽 하단에 표시됩니다. 해당 리소스가 초과되면 빨간색으로 강조 표시됩니다.

창 하단의 에너지 아이콘을 클릭하면 리액터 추가를 위한 빠른 메뉴가 나타납니다.

방 경계

방을 다른 방에 너무 가깝게 이동하여 경계가 겹치는 경우 빨간색으로 강조 표시됩니다. 방 경계가 겹치는 한 도장 레이아웃은 유효하지 않습니다.

인접한 층의 방도 경계와 겹칠 수 있습니다. 현재 보이는 방이 다른 층의 방과 겹치는 경우 다른 방의 경계만 빨간색으로 표시됩니다.

정박소에 대한 특별 참고사항

정박소는 매우 큽니다. 정박소에서 보이는 윤곽선과 경계는 후면까지 완전히 확장되지 않으며, 이는 정박소의 눈에 보이는 크기보다 대략 6~7배 더 확장됩니다. 정박소의 전체 길이에 대한 경계 충돌은 여전히 감지되지만, 정박소를 축소하여 찾지 않으면 무엇이 충돌하는지 명확하지 않을 수 있습니다.

단종 방에 대한 특별 참고사항

오래되어 단종된 도장 방을 레이아웃에 배치할 수 있습니다. 경고 아이콘이 함께 제공됩니다. 주의할 사항이 많기 때문입니다.

나는 이전 방 경계를 테스트하는 데 도움을 받았습니다(x-cessive에게 큰 감사를 드립니다).

하지만 테스트를 하더라도 단종된 방의 정보는 다른 방보다 정확하지 않습니다. 표준 격자를 따르지 않습니다. 그리고 도장에 단종된 방이 없기 때문에 지속적으로 테스트할 수는 없습니다. 그렇기에 몇 더 가지 가정을 해야합니다.

  • 오래된 원자로가 새로운 원자로와 동일한 25 에너지를 제공한다고 가정합니다.
  • 다른 모든 것에서는 에너지를 사용하지 않고 용량도 소비하지 않는다고 가정합니다.
  • 모든 단종된 방이 스타트 클랜 홀의 높이와 깊이를 가지고 있다고 가정합니다.

특정 방 규칙

특정 방에는 다음과 같은 몇 가지 제약이 적용됩니다.

  • 최소한 하나의 입장 방이 필요합니다. 이는 방 메뉴에 생성 지원 아이콘으로 표시되어 있습니다. 설정이 가능한 방은 다음과 같습니다:
    • 모든 클래식 클랜 홀
    • 영감의 홀
    • 오로킨 엔트라티 접견실
    • 인페스티드 방
    입장 방에 대한 자세한 내용은 아래를 참조하세요.
  • 오라클은 모든 연구실의 전제 조건입니다.
  • 각 클래식 클랜 홀은 하나씩만 보유할 수 있습니다.
  • 영감의 홀은 최대 3개까지만 보유할 수 있습니다.
  • 장애물 코스 설계자 방은 최대 5개까지만 보유할 수 있습니다.
  • 병영은 각 유형당 하나만 보유할 수 있으며, 각 등급의 병영은 다음 병영의 전제 조건입니다.
  • 정박소는 하나만 가질 수 있습니다.

방이 특정 규칙을 위반하면 빨간색으로 강조 표시됩니다.

기타 제한사항 및 피드백

방이 리소스, 경계 또는 규칙 제한 조건을 위반하면 빨간색으로 강조 표시됩니다. 방을 두 번 클릭하면 방 메뉴가 나타나는데 방 이름 옆에는 오류 버튼이 있습니다. 그것을 클릭하면 무엇이 잘못된 것인지 알 수 있는 제한 조건 목록이 나타납니다.

화면 왼쪽에 있는 층 표시기는 제한 조건 위반이 있는 방이 포함된 모든 층을 빨간색으로 강조 표시합니다.

방 추가 메뉴에서 방을 클릭해서 추가하려고 할 경우, 제한조간이 발생하고 있는 방의 오른쪽에는 오류 버튼이 표시됩니다. 이는 개별 방 메뉴의 복제 옵션에도 적용됩니다.

마지막으로 상단 표시줄에는 확인, 오류 또는 경고와 같은 전역 오류 아이콘이 있습니다. 이를 클릭하면 도장에서 감지된 모든 오류 또는 경고 목록이 표시됩니다.

자원

방을 추가하거나 복제할 때 에너지 및 용량 제한 옆에 리소스 버튼이 표시됩니다. 이를 클릭하면 현재 클랜 티어를 고려하여 해당 방을 건설하는 데 필요한 자원이 표시됩니다.

화면 하단의 총 공간, 에너지 및 용량 옆에는 또 다른 자원 버튼이 있습니다. 이를 클릭하면 현재 클랜 티어를 고려하여 전체 도장을 건설하는 데 필요한 총 자원이 표시됩니다.

클랜 티어

클랜 등급은 존재하는 가장 높은 등급의 막사 방에 따라 결정됩니다. 5가지 계층이 있습니다.

  1. 0. 고스트 (배럭 없음)
  2. 1. 새도우
  3. 2. 스톰
  4. 3. 마운틴
  5. 4. 문

현재 클랜 등급은 화면 하단의 클랜 선호도 및 자원 옆에 표시됩니다. 현재 레이아웃에서 누락된 배럭들의 목록을 보려면 이 아이콘을 클릭하세요.

보통 당신이 배럭을 짓기 전에 최소한 도장에 방 몇개는 이미 지어둔 상태일것이기 때문에, 표시된 자원의 총합이 틀릴수 있습니다. 표시된 모든 자원은 레이아웃의 '현재' 클랜 등급에 대한 것입니다.

클랜 숙련도

지을 수 있는 일부 방은 처음 지을 때 클랜 숙련도를 제공합니다.

클랜 숙련도 방

현재 클랜 숙련도를 제공하는 12개 방은 다음과 같습니다.

  • 오라클
  • 7개 연구소 모두
  • 두 종류의 장애물 코스
  • 결투장
  • 영광의 사원

이 방은 첫 번째 방이 레이아웃에 추가될 때까지 메뉴에 클랜 XP를 제공하는 방 아이콘과 함께 표시됩니다. 그 후에는 화면 하단에서 클랜 XP 수를 제공하는 방에 추가됩니다.

화면 하단에 있는 클랜 숙련도 방 아이콘을 클릭하면, 현재 레이아웃에서 누락된 모든 클랜 숙련도를 얻을 수 있는 방을 보여주는 메뉴가 나타납니다.

참고 : 방들을 하나 이상 더 건설해도 숙련도는 추가적으로 얻을 수 없습니다. 한 종류당 한번씩입니다.

참고 : 실제 도장에서 방을 삭제하면 다시 지을때까지 해당 방 만큼의 클랜 숙련도를 잃게됩니다.

저장 및 공유

현재 레이아웃과 보기는 브라우저의 위치 표시줄에 지속적으로 저장됩니다. 브라우저 기록에서 앞으로 또는 뒤로 이동하는 경우 이를 반대로 하면 이전에 가졌던 레이아웃과 보기가 복원됩니다.

공유

메뉴, 공유, 복사를 차례로 클릭하여 브라우저 위치의 복사본을 사용할 수도 있습니다. 이렇게 하면 클립보드에 현재 레이아웃에 대한 링크가 배치됩니다. 누구나 이 링크를 사용하여 당신의 도장의 레이아웃을 만들 수 있습니다.

로컬 저장소

메뉴를 클릭한 다음 로컬 저장소를 클릭하여 로컬 저장소를 엽니다.

추가를 사용하여 현재 레이아웃을 로컬 저장소의 새 항목으로 추가합니다. 레이아웃에 이름을 지정하라는 메시지가 표시됩니다.

로컬 저장소에 기존 항목이 있는 경우, 저장을 사용하여 해당 항목을 현재 레이아웃으로 덮어쓸 수 있습니다.

. 삭제를 사용하여 기존 항목을 제거할 수 있습니다.

해당 열을 기준으로 정렬하려면 이름 및 날짜 열 머릿글을 클릭하세요. 다시 클릭하면 정렬 방향이 반전됩니다.

참고: 로컬 저장소는 당신의 각 브라우저에 저장됩니다. 레이아웃을 다른 사람과 공유하거나 다른 브라우저에서 자신과 공유하려면 공유를 사용하여 링크를 공유해야 합니다.

PNG로 내보내기

메뉴, PNG를 차례로 클릭하여 도장 레이아웃을 하나 이상의 PNG 파일로 내보내기 위한 메뉴를 불러옵니다.

  • 배율 필드는 PNG 이미지의 배율을 제어합니다. 미터당 픽셀 단위입니다. 스케일이 클수록 이미지도 커집니다.
  • 크기 필드에는 생성될 PNG의 크기가 표시됩니다.

생성을 클릭하여 PNG 이미지를 생성합니다. 다운로드할 수 있는 링크로 표시됩니다.

도장에 여러 층이 있는 경우 각 층마다 별도의 이미지가 생성됩니다. 각 층의 이미지에는 동일한 크기와 위치가 사용되므로 서로의 위에 정렬됩니다.

큰 도장은 큰 이미지를 생성하기 때문에 이미지 링크가 나타나는 데 약간의 시간이 걸릴 수 있습니다.

입장 지점

어느 방에서나 방 메뉴를 클릭하고 입장 지점 설정을 선택하여 해당 방을 입장 지점이 포함된 방으로 지정할 수 있습니다. 다른 방에 입장 지점이 있는 경우 저짐이 이동됩니다.

참고: 입장 지점은 입장 방과 다릅니다. 도장에는 트리 구조의 루트를 결정하는 입장 방이 있어야 하며, 특정 방만 입장 방이 될 수 있습니다. 그러나 입장 지점은 선택 사항이며 모든 객실에 배치할 수 있습니다.

입장 지점이 있는 방에는 입장 지점 마커가 있습니다.

레이아웃에서 입장 지점를 제거하려면 방 메뉴를 클릭하고 입장 지점 제거를 선택하세요. 이후에는 레이아웃에 입장 지점이 제거됩니다.

구조 모드

기본 모드

기본 구조 검사는 기본적으로 활성화되어 있습니다. 당신이 해야 할 일은 입장 방을 추가하는 것 뿐입니다.

입장 방

입장 방 기능을 지원하는 방에서는 방 메뉴를 클릭하고 입장 방 설정을 선택하여 해당 방을 입장 방으로 지정합니다. 다른 방이 이미 입장 방으로 지정되어 있는 경우 입장 방은 이동됩니다.

참고: 입장 방은 입장 지점과 다른 기능입니다.

지정된 방에는 입장 방 마커가 표시됩니다.

레이아웃에서 입장 방을 제거하려면 방 메뉴를 클릭하고 입장 방 삭제 입장 방 제거를 선택하세요. 이후에는 레이아웃에서 입장 방이 제거됩니다.

연결

입장 방이 지정되면 모든 방이 직접, 또는 다른 방을 통해 간접적으로 입장 방 에 연결되어 있는지 확인하기 위해 도장을 확인합니다.

입장 방에 연결되지 않은 방은 노란색으로 강조 표시되고 경고로 표시됩니다.

고급 모드

메뉴, 구조를 차례로 클릭한 다음 활성화됨 확인란을 선택하여 고급 구조 모드를 활성화합니다.

고급 구조 모드를 활성화하려면 입장 방이 필요합니다.

고급 구조 모드가 활성화되면 도장 레이아웃이 녹색 선으로 오버레이됩니다. 이는 게임 내에 존재하는 도장의 내부 구조를 보여줍니다.

당신의 도장은 나무와 같습니다

방을 도장에 추가할 때마다 문부터 시작합니다. 새 방을 만드는 데 사용하는 문은 양쪽 방 사이에 부모-자식 관계를 만듭니다. 처음 만든 방은 부모이고, 이어지는 두 번째 방은 자식입니다. 이 빌드 순서는 대부분 확정되어 있습니다. 그 방이 파괴될 때까지 이 내부설정은 고정됩니다.

어떤 방에서 시작하든 순서대로 방을 이어서 지으면 자식방에서 자식방으로, 또 자식방으로 이어지며 나무가 뿌리에서 가지를 뻗듯이 이어집니다. 컴퓨터 과학에는 이러한 종류의 구조에 대한 기술 용어가 있습니다. 이것을 "트리"라고 합니다. 컴퓨터공학자들은 이름 짓는 걸 잘 못해요.

이 트리의 시작점. 즉 뿌리는 입장 방입니다. 입장 방에서 시작하는 분기에는 그 안에 다른 모든 분기가 포함됩니다.

고급 구조 모드가 활성화되면 DojoCAD는 생성 공간부터 시작하여 레이아웃에서 트리 구조를 구축하려고 시도합니다. 레이아웃에 루프가 포함되어 있지 않으면 자동으로 그렇게 할 수 있습니다.

트리 구조 표시

트리 구조는 고급 구조 모드에서 문 위에 녹색 화살표로 표시되며, 부모 방에서 자식 방으로 이어져나갑니다. 이 화살표는 각 방의 중앙을 통과하는 선으로 연결됩니다. 각 방의 중앙은 더 큰 녹색 사각형으로 강조 표시됩니다.

이후 선택

고급 구조 모드가 활성화되고 레이아웃에 유효한 트리 구조가 있으면, 방 메뉴에 방 선택을 위한 몇 가지 추가 옵션이 표시됩니다.

방 한개를 선택하고 이후를 사용하면 해당 방에서 분기되는 모든 자식 방이 표시됩니다.

게임 내에서 방 아래의 분기는 입장 방이 동일하게 유지된다는 가정하에, 해당 방이 파괴되기 전에 파괴되어야 하는 모든 방을 나타냅니다.

이전 선택

방을 한개를 선택하고 이전을 사용하면 해당 방에서 입장 방으로 연결되는 모든 상위 방의 경로가 표시됩니다.

게임 내에서 입장 방으로 이어지는 경로는, 입장 방이 동일하게 유지된다는 가정하에 해당 방을 건설하기 전에 건설해야 하는 모든 방을 나타냅니다.

입장 방 설정 이동

레이아웃에 유효한 트리 구조가 있는 경우 입장 방은 어디든 다시 설정할 수 있으며, 트리구조는 여전히 유효하게 유지됩니다.

입장 방이 트리를 따라 방의 한쪽에서 다른 쪽으로 이동하면 해당 방의 가지 구조가 변경됩니다. 이전 부모는 자식 분기 중 하나가 되고, 이전 자식 방 중 하나는 부모 방이 됩니다. 이는 때때로 이전 자식 방 중 하나를 보존하면서 방을 파괴하는 데 도움이 되지만, 실제로는 수많은 다른 방을 파괴하는 것을 피하는 것이 꽤 어렵습니다.

루프

나무의 특징은 가지가 뻗은 후에 다시 연결되지 않는다는 것입니다. 트리의 구조가 유효하다면 이미 방문했던 방을 되돌릴 수 있는 방법이 없습니다. 어쨌든 궁극적으로 분기는 끝이 있어야 합니다.

그런데 우리는 분명 루프가 포함된 도장을 만들 수 있습니다. 대체 무슨 일일까요?

게임을 해보면 다른 맵에는 루프가 없다는 것을 알 수 있습니다. 게임 내 임무에서 지도를 탐색하면 막다른 골목만 찾을 수 있습니다. 여러 방을 포함하는 대규모 루프가 없습니다. 도장에만 그런 것이 있을 수 있는 것 같습니다.

나는 내부적으로는 미션 맵과 도장을 모두 구동하는 동일한 기본 엔진이라고 확신합니다. 둘 다 트리 구조를 사용하여 분기 측면에서 레이아웃을 정의합니다. 그러나 도장에는 두 가지 사이를 지나갈 수 있는 특별한 종류의 문이 있습니다. 저도 이름을 잘 못정해서 이걸 교차분기라고 부릅니다.

루프 감지

고급 구조 모드가 활성화되면 감지된 모든 루프가 두 가지 방식으로 표시됩니다. 루프에 관련된 방은 노란색 경고로 강조 표시됩니다. 루프와 관련된 문에는 양방향 노란색 화살표가 표시됩니다.

레이아웃이 복잡하다면 일부 룸이 하나 이상의 루프에 포함될 수 있다는 점에 유의하세요.

이전 이후 선택과 같은 대부분의 고급 구조 모드 기능은 루프가 해결될 때까지 사용할 수 없습니다.

교차분기

문 위에 있는 화살표를 클릭하면 작은 메뉴가 나타납니다. 해당 문을 강제로 교차분기로 만들려면 교차분기를 사용하세요.

교차분기는 연결이 허용되지만 레이아웃의 트리 구조를 결정할 때 계산되지 않습니다.

교차분기를 클릭하고 재설정을 사용하여 교차분기 설정을 재설정합니다.

방을 이동하거나 삭제하여 문 연결을 끊으면 해당 문에 있을 수 있는 모든 교차 지점 설정도 지워집니다.

교차분기를 찾는 방법

DojoCAD를 사용하여 기존 도장을 모델링하는 경우 교차분기들은 도장에 이미 설치되어 있습니다. 게임에 들어가서 그들이 어디에 있는지 찾으면 됩니다. 이것은 지루할 수 있지만 어렵지는 않습니다..

방을 파괴하려고 하면 다음 두 가지 중 하나가 발생합니다.

  1. 정말로 파괴를 위해 방을 대기열에 넣을 것인지 묻는 메시지가 나타납니다. 이는 귀하가 있는 방이 지점의 끝에 있고 자식 방이 없음을 의미합니다.
  2. 자식 구성요소에 대한 메시지와 이를 미니맵에 표시하는 옵션이 표시됩니다. 그러면 게임 내 지도에 파란색 마커로 자식 방이 강조 표시됩니다.

실제로 방을 파괴할 필요는 없습니다. 두 가지 메시지 중 하나를 볼 수 있을 만큼 멀리 가는 것만으로도 충분합니다.

DojoCAD를 사용하여 기존 도장을 모델링하는 경우 교차분기들은 도장에 이미 설치되어 있습니다. 게임에 들어가서 그들이 어디에 있는지 찾으면 됩니다. 이것은 지루할 수 있지만 어렵지는 않습니다.

양쪽의 방을 파괴하려고 할 때 강조 표시되지 않는 문을 발견하면 그것이 교차분기입니다.

이를 실제로 적용하면 다음과 같습니다.

  1. DojoCAD에서 반복 경고가 있는 도장의 방을 선택하고 이를 파괴해 보십시오. 입장 방이 아니라요.
    • 입장 방에서 이 작업을 수행하면 많은 교차 지점 문을 빠르게 찾을 수 있을지도 모르지만, 이 기능이 작동하지 않는 경우가 많습니다. 입장 방이 아닌 다른 방에서 시작한 후 나머지 바법을 수행하는 것이 모든 경우의 수를 확실히 알 수 있습니다.
  2. 즉시 파괴 대기열에 넣으라는 메시지가 표시되면 루프를 따라 문을 선택하고 4단계로 이동합니다.
  3. 자식 방 프롬프트가 표시되면 자식 방을 표시하도록 지시하세요.
  4. 콘솔에 아이콘이 있는 방에 도달할 때까지 또는 루프의 다음 문에 아이콘이 없는 방에 도달할 때까지 파란색 아이콘이 있는 문을 통해 루프를 따라갑니다.
  5. 저 문으로 가서 그 방을 파괴해 보세요.
  6. 즉시 파괴 대기열에 넣으라는 메시지가 표시되면 8단계로 이동합니다.
  7. 자식 방 프롬프트가 표시되면 자식 방을 표시하도록 지시하세요. 방금 통과한 문에는 파란색 아이콘이 있어서는 안 됩니다.
  8. DojoCAD에서 방금 통과한 문을 교차분기로 표시하세요.
  9. DojoCAD가 더 이상 루프를 표시하지 않을 때까지 반복합니다.

5, 6, 7단계가 항상 필요한 것은 아니지만 직접 확인해 보는 것이 좋습니다.

모든 층 표시

기본적으로 한 층을 보는 동안 다른 모든 층은 배경에서 회색으로 표시됩니다. 너무 복잡해지거나 브라우저가 지연되는 경우 모든 층 표시 설정을 사용하여 비활성화할 수 있습니다.

모든 층 표시를 비활성화한 경우에도 Ctrl+Shift+A, Ctrl+Shift+드래그 및 동일한 유형이나 색상의 방을 선택하는 다양한 방법을 사용하여 다른 층의 방을 선택할 수 있습니다. 다른 방은 선택한 경우에만 표시됩니다.

지도 마커 표시

기본적으로 실험실 및 정박소와 같은 특정 방의 지도 마커가 표시됩니다. 이 기능을 끄려면 지도 표시 표시 설정을 사용하세요.

이 설정은 PNG 기능으로 생성된 이미지에도 적용됩니다.

라벨 표시

편집할 때 라벨이 복잡해지는 경우가 있습니다. 라벨 표시 설정을 사용하면 라벨을 제거하지 않고 비활성화할 수 있습니다.

이 설정은 PNG 기능으로 생성된 이미지에도 적용됩니다. 또한 라벨 표시가 활성화되어 있지 않으면 라벨을 편집할 수 없습니다.

방 어둡게 보기

때로는 밝은 방 색상으로 인해 라벨을 읽거나 고급 구조 모드를 보는 것이 더 어려워질 수 있습니다. 실제 방을 더 어둡게 만들려면 방 어둡게 보기 설정을 사용하십시오.

솔직히 말하자면. 이 설정은 고급 구조 모드를 멋있게 보이게 하기 위해 넣었어요.

규칙 활성화

이 설정을 사용하여 방 제한, 전제 조건, 에너지 및 용량에 대한 규칙을 해제합니다. 비활성화하면 해당 사항이 확인되지 않으며 방 및 레이아웃에 오류가 추가되지 않습니다.

간섭 오류는 여전히 출력됩니다.

자동 저장

자동 저장 설정이 활성화된 경우, 레이아웃을 편집하면 로컬 저장소의 고유 슬롯에 자동으로 저장됩니다.

링크에 레이아웃 없이 DojoCAD를 열면 현재 자동 저장으로 저장된 모든 항목이 자동으로 로드됩니다.

자동 저장된 레이아웃이 있지만 실수로 레이아웃이 포함된 DojoCAD 링크를 로드한 경우, 실제로 편집할 때까지 원래 자동 저장된 레이아웃은 로컬 저장소에 그대로 남아 있습니다.

언어

DojoCAD가 브라우저에 설정된 기본 언어를 지원하는 경우 자동으로 적용되어야 합니다. 다른 언어를 선택하려면 메뉴, 언어를 차례로 클릭하고 언어 메뉴를 엽니다.

그러면 현재 DojoCAD에서 지원하는 모든 언어가 표시됩니다. 하나를 선택하면 즉시 적용됩니다. 이 설정은 브라우저에 저장되므로 DojoCAD를 다시 열면 이전에 선택한 것과 동일한 언어가 자동으로 사용됩니다.

참고: 누군가 자원 봉사를 원하지 않는 한 모든 번역은 Google 번역을 통해 수행됩니다. 정확성에 대해서는 어떠한 보장도 할 수 없습니다.
